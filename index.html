Astra-OS
MVP

A local-first personal AI operating system. No cloud. No surveillance. Every inference runs on-device. Astra-OS gives you a persistent, reflective AI agent that knows your history, reasons over your tasks, and builds context over time.

OTPAR Agent Loop
Observe Think Plan Act Reflect
⚡
Modular Tool Architecture
FastAPI backend with structured tool-calling enabling composable agent execution across arbitrary task types.
🧠
Vector Memory
ChromaDB-powered persistent contextual recall — the agent remembers and retrieves across sessions.
🔄
Reflection-Based Retry
Self-correcting task execution via the Reflect step — the agent audits its own output and retries on failure.
🔒
Fully Local — Privacy First
Runs on Ollama and WebLLM. Zero cloud dependency. Your data never leaves your device.
